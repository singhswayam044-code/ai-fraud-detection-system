#  AI-Driven Fraud Detection System

A hybrid fraud detection system combining Machine Learning, Graph Neural Networks (GNN), and Temporal Analysis (LSTM) to detect fraudulent transactions with high accuracy and interpretability.

---

##  Features

- Hybrid AI Model (ML + GNN + LSTM)
- Risk-Based Classification (Low / Medium / High)
- Fraud Network Visualization
- Explainable AI (Feature + Graph + Anomaly insights)
- Automated Fraud Report (CSV)
- Interactive Dashboard (Streamlit)

---

##  Architecture
Data → Preprocessing → ML Models → LSTM → GNN → Hybrid Decision → Risk Classification → Dashboard

---

## Models Used

- Isolation Forest → Anomaly Detection  
- Random Forest → Classification  
- Logistic Regression → Probability Scoring  
- LSTM → Temporal Pattern Detection  
- GNN → Relationship-Based Fraud Detection
